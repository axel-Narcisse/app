Element html

Tag Selects all elements of type A. Type refers to the type of tag, so div, p and ul are all different element types.

div { #parametreCss} selects all div elements.
ID

#idSelects the element with a specific id. You can also combine the ID selector with the type selector.

#cool { #parametreCss} selects the unique ul id="cool" elements.
